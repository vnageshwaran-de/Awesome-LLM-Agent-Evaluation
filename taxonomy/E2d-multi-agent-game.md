# E2d — Multi-agent / game worlds

**Pillar:** III — WHERE it is tested (Environment Topologies; dynamic E2)

**Definition.** Simulated multi-agent environments and game worlds for coordination, negotiation, and competition (W5) under controlled, fully observable, resettable dynamics.

**Strength.** Clean experimental manipulation and deterministic outcome scoring.

**Limitation.** Reduced ecological validity — the gap between a constrained game world and an open social or economic setting is the external-validity cost of experimental control.

**Benchmarks:** AgentBench (game environments).
