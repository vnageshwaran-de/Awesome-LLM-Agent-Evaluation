# E2b — Web / browser containers

**Pillar:** III — WHERE it is tested (Environment Topologies; dynamic E2)

**Definition.** Containerized or live web environments for navigation, form-filling, retrieval, and transaction tasks over rendered pages.

**Central tension.** *Live vs frozen* — live sites maximize realism but destroy reproducibility (content drifts, sites change, network failures contaminate results); frozen mirrors restore reproducibility at the cost of staleness and a narrowed task distribution.

**Security note.** Principal vector for adversarial perception attacks (W6 via injected page content).

**Benchmarks:** WebArena (self-hosted functional sites); GAIA (open-web solving component).
